# CrazeOS
Simple work-in-progress amateur OS written in C.

# Features
- P2/2 Keyboard Input
- Functional shell
- VGA Buffer
- (Interrupts)

# Build from source
Execute ```cd src/ && make && make run``` to get started!
- LINUX SUPPORTED ONLY AT THE MOMENT -
